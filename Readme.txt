## Basic commands while running through the project

## Command to activate a Python virtual environment on Windows operating systems.
venv\Scripts\activate

## Command to deactivate a Python virtual environment on Windows operating systems.
venv\Scripts\deactivate

## Command is used to create a Python virtual environment. This command utilizes the built-in venv module in Python.
python -m venv env

## The command "pip freeze > requirements.txt" is used to generate a requirements.txt file that lists all the Python packages 
and their versions installed in the current Python environment

## To execute the project use command
python app.py

And enter the url the one which shows on the browser tab (for ex:-https://www.youtube.com/watch?v=dagW7odcxfE)
